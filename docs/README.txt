:: README ::

> to run the project open a console in the current directory and type the following command:

		"java -jar app-0.0.1-SNAPSHOT.war"
		
> report file can be found in "docs" folder
		
		
		
:: IMPORTANT ::

> a PostgreSQL user with username "postgres" and password "postgres"
	and a database called "projdata" are needed to be able to run the project
	
> for testing purposes, User table and its creation pages are "unlocked" and can be accessed by anyone
	this is made to make it easier to create a first admin account
